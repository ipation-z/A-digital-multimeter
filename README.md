# A-digital-multimeter
A simple digital multimeter based on STC8H8K64U

项目简介

本项目设计并实现了一款功能齐全的3½位数字万用表，主控芯片采用STC8H8K64U（增强型51内核），具备直流电压、电阻、频率/占空比、通断和二极管测量功能。系统通过单按键实现功能切换和量程调节，采用LCD1315显示测量结果。本仓库用于留存该工程，仅供学习参考。

主要特性
测量功能：

直流电压：2000mV / 20V 两档

电阻：200Ω / 2kΩ / 20kΩ / 200kΩ 四档

频率/占空比：20Hz / 200Hz / 2kHz / 20kHz / 200kHz 五档，占空比分辨力1%

通断测量：<10Ω 连续蜂鸣

二极管：显示极性及导通压降

操作方式：单按键，短按切换量程，长按切换功能

显示：LCD1315字符液晶（I2C接口）

精度：精度10%左右，可后续升级硬件或软件校准

供电：USB 5V供电
